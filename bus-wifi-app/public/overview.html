SEC 05 Developer Instructions — Antigravity Tool Backend Rules
⚙ Labor Cost Scaling Rule
If Camera_Qty <= 2 → Labor_Cost = 1,500
If Camera_Qty >= 3 → Labor_Cost = 2,000
Scale automatically based on camera count at time of invoice generation.
💳 eSewa Cashback Subtraction
On every renewal event, auto-subtract cashback from telecom cost variable:
Basic: − Rs. 117 → Net = Rs. 4,883
Premium: − Rs. 188 → Net = Rs. 7,812
🔀 1-Year Premium Split Logic
When SKU 1-Year 10Mbps Premium (Rs. 25,000) is sold, project backend costs sequentially:
Phase 1 → Initial 10Mbps (Rs. 9,000)
Phase 2 → Renewal 5Mbps (Rs. 4,883)
🗑 Hardware Depreciation on Renewal
For any Phase 2+ renewal invoices, drop all hardware costs:
No routers · No accessories · No initial setup fees.
Calculate using Net Renewal Costs only.
